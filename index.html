NIGHTTIME RUNNERS - VOIDEA

(10/03/23 - Ongoing)

ABOUT THE GAME

Nighttime Runners Is an asymmetric multiplayer “capture the flag”-like game, played at night! Forty stealthy terrain-traversal enthusiasts (Runners) attempt to claim their package and return to base, while eight tracking techies with flashlights (Seekers) attempt to catch them.


THE TEAM

Voidea is a twelve person indie studio made up of CG Spectrum students/alumni. I work with two other programmers and the designers to implement new features in C++ and Blueprints in Unreal Engine 5.

  • Joined Nighttime Runners halfway through development on 19/06/23.
  • Worked with replication within multiplayer systems.
  • Programmed a gear system involving data tables for gear effects and cosmetics.
  • Implemented a dynamic spawning system for forty eight players.
  • QA'd and bug-fixed packaged builds.
  • Maintained and implemented an external plugin: EIK, Epic (Online Services) Integration Kit. Links: EIK Source and EOS Setup Guide

Runners will spawn in one of six possible spawn points and the flag/package will spawn in a possible three opposite points.

QNN

(03/04/23 - Ongoing)

ABOUT THE GAME

First person puzzle game in a dystopian future where an AI known as QNN (Quantum Neural Networks) has destroyed the world. You play as a cyborg who's arms have been replaced with robotic limbs each with their own abilities. The player must escape from the facility and traverse through a hazardous environment.


  • Game Design Document:
  • GitHub:
  • QNN GDD
  • ChaiChai676/QNN

THE TEAM

I collaborated with various CG Spectrum students from 3D animation, 3D modelling, game design and UI/UX design to bring to life their ideas on this Unreal Engine 5 project.

  • Made an ability system for binding and unbinding Input Actions using Enhanced Inputs.
  • Worked with the UI/UX designer to animate UMG Widgets (health bars, mini-maps).
  • Created a flexible movement and spotlight component which can be switch activated.
  • Used Gameplay Tags to change interactions for objects of varying properties.
  • Implemented a damage system as well as health pickups and respawning.
  • Retargeted UE mannequin's base animations onto a custom model.
  • Used blend spaces to match animation based on character speed.
  • Worked with a 3D animator to create a Zelda inspired grapple.
  • Onboarded new team members for the project.

SUBMERGE

(19/01/23 - 02/03/23)

ABOUT THE GAME

A first person survival puzzle game completed as part of CG Spectrum's 6 week game jam where participants are paired with other members of the community. Each team makes up at least one programmer, designer and 3D modeller.


Challenge rules: a game level has to be completed in 6 weeks with only 10 unique art assets and at least 2 interactions.

THE TEAM

For this challenge I liaised with two game designers and a programmer to build this Unreal Engine 5 project under the challenge rules and time restriction.

  • Taught version control to the team and how to set up and reset projects with Visual Studios.
  • Implemented line traces and debug messaging from Kismet library.
